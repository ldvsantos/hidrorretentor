# Industrial Crops and Products — submission package checklist

## Files to upload (typical Elsevier/ICP workflow)

### Mandatory / strongly expected
- **Manuscript (DOCX)**: `Hidrorretentor_Taboa_EN.docx`
- **Highlights (editable file)**: `Hidrorretentor_Taboa_highlights.txt`
- **Cover letter**: `Hidrorretentor_Taboa_cover_letter_ICP.md` (copy/paste into the submission portal or upload if allowed)

### Optional / if applicable
- **Graphical abstract**: prepare and upload if you have a suitable figure (see `Hidrorretentor_Taboa_graphical_abstract_BRIEF.md`).
- **Supplementary material**: only if you want to provide raw tables/extra figures beyond the main manuscript.
- **Suggested reviewers**: if the submission system requests, fill `Hidrorretentor_Taboa_suggested_reviewers_TEMPLATE.md`.

## Content already inside the manuscript
- Data availability statement includes Zenodo DOI.
- Declaration of competing interests.
- CRediT authorship contribution statement.
- Funding statement.
- Acknowledgements.

## Quick pre-submit sanity check
- Generate DOCX via `python gerar-docx.py` and confirm: no Pandoc errors, figures appear, references render.
- Confirm Highlights bullets are ≤85 characters each.
- Confirm Abstract ≤250 words.
