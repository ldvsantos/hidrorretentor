# Núcleo hidrorretentor orgânico como substrato para bioengenharia dos solos

## Resumo
Este repositório contém os dados, scripts e manuscritos relacionados ao estudo de núcleos hidrorretentores orgânicos à base de *Typha domingensis* para bioengenharia de solos.

**Objetivo:** Avaliar um núcleo hidrorretentor orgânico produzido com fibras e resíduos de *Typha domingensis* como substrato para cultivo de rúcula (*Eruca sativa*), integrando caracterização tecnológica, triagem biológica e desempenho em microcosmo.

## Conteúdo

- `1 - OLD/`: Versões antigas e dados legados.
- `2 - DADOS/`: Dados brutos experimentais (SPSS, CSV, etc.).
- `3 - MANUSCRITO/`: Código fonte do manuscrito (Markdown, scripts de conversão, bibliografia).
- `5 - FIGURAS/`: Imagens e figuras geradas para o artigo.

## Reproducibilidade

Os scripts de análise estão localizados em `3 - MANUSCRITO/1-MARKDOWN/3-SCRIPTS/`.
